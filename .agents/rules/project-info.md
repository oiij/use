# Project Information

## Overview

- **Project Name**: @oiij/use-monorepo
- **Type**: Vue 3 Monorepo (pnpm workspaces)
- **Description**: A monorepo for Vue 3 composables, components and utilities
- **Author**: oiij
- **License**: MIT

## Repository

## Key Commands

- `pnpm docs:dev` - Start VitePress dev server
- `pnpm pkg:build` - Build all packages
- `pnpm test` - Run Vitest
- `pnpm lint` - Run ESLint
- `pnpm type:check` - Run vue-tsc
