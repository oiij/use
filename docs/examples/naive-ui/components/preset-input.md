## Props

| Name    | Type               | Default | Description |
| ------- | ------------------ | ------- | ----------- |
| value   | String             | -       | 输入的值。  |
| path    | String             | -       | 输入路径。  |
| options | PresetInputOptions | -       | 输入框配置  |

## Emits

| Name         | Type   | Default | Description |
| ------------ | ------ | ------- | ----------- |
| update:value | String | -       | 输入的值。  |
